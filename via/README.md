<h1>Projeto - Teste Selenium</h1>
<H2>Para executar o projeto:</H2>

<p>
1.Importar o arquivo para um projeto Maven; <br/>
2.Instalar as dependencias; <br/>
3.Executar a feature HU01-SolicitarCotacao.feature no caminho /via/src/test/resources/features/

Obs:
Para escolher o browser, abrir a classe Propriedades.java no caminho /via/src/test/java/helper/ e alterar o navegador para o desejado:
	public static Browsers browser = Browsers.<b>CHROME</b>;



Os drivers estão na pasta via/drivers e correspondem às versões a seguir:<br/>
 Chrome - versão 87 <br/>
 Firefox - versão 82
 
 </p>
